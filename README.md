# Student-Registration-System

Student Registration System built with HTML, CSS, and JavaScript. Add, edit, delete, and view student records with validation and localStorage used.

## Github

**https://github.com/raju-kadachha/Student-Registration-System**

## Features

- Add a new student with Name, ID, Email, and Contact Number.
- Edit existing student details.
- Delete student records with confirmation.
- Input validation:
  - Name: only letters and spaces
  - Student ID: exactly 6 digits
  - Email: valid email format
  - Contact: 10-digit Indian mobile number
- Responsive layout for desktop, tablet, and mobile.
- Sticky table headers for easy reading.
- Notification messages for add, edit, delete, and errors.
- Data saved in browser using localStorage (remains after page reload).
- Table scroll appears automatically if many students are added.

## Files

## How to Use

1. Download or clone the project files.
2. Open `index.html` in any web browser.
3. Use the form to add student details.
4. Edit or delete records as needed.
5. Data remains saved even after page refresh.
